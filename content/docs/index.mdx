Abui is a custom UI component registry built on top of [shadcn/ui](https://ui.shadcn.com/). It provides a collection of beautifully designed, accessible, and customizable React components that you can copy and paste into your projects.

## Getting Started

To use components from the Abui registry, first configure your `components.json`:

```json
{
  "registries": {
    "@abui": "https://abui.antoniobrandao.com/r/{name}.json"
  }
}
```

Then install components using:

```bash
npx shadcn@latest add @abui/button
```
