## Features

- **Email validation** — Built-in email format validation
- **Password field** — Secure password input with proper type
- **Remember me** — Optional checkbox for persistent sessions
- **Forgot password** — Link to password recovery flow
- **Social login** — Ready for OAuth provider integration
- **Responsive** — Works on all device sizes
- **Accessible** — Proper ARIA labels and keyboard navigation

## Customization

The login form uses your application's theme tokens and can be customized via CSS variables or by modifying the component directly.

### Adding OAuth Providers

You can easily add social login buttons:

```tsx
<LoginForm>
  <div className="grid grid-cols-2 gap-4">
    <Button variant="outline">
      <GithubIcon className="mr-2 h-4 w-4" />
      Github
    </Button>
    <Button variant="outline">
      <GoogleIcon className="mr-2 h-4 w-4" />
      Google
    </Button>
  </div>
</LoginForm>
```

### Form Handling

Integrate with your authentication system:

```tsx
function LoginPage() {
  const handleSubmit = async (data: LoginData) => {
    // Your authentication logic here
    await signIn(data)
  }
  
  return <LoginForm onSubmit={handleSubmit} />
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `onSubmit` | `(data: LoginData) => void` | - | Form submission handler |
| `showRememberMe` | `boolean` | `true` | Show remember me checkbox |
| `showForgotPassword` | `boolean` | `true` | Show forgot password link |
